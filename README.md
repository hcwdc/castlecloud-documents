# cdoc

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father-build`,

```bash
$ npm run build
```

## 本工程介绍

本工程为将我们的对外开放类的文档全部搬家整理至这个平台上，真正做到可查文档开发。
