# 节假日查询

**简要描述：**

- 本接口可以查询节假日、加班日期，以及每个日期的对应的国际日和我国传统节日的简介。

**请求 URL：**

- ` http://api.icuapi.com/openapi/life/legalHolidays`

**请求方式：**

- Get

**返回类型**

- JSON

**参数：**

| 参数名 | 必选 | 类型   | 说明                    |
| :----- | :--- | :----- | ----------------------- |
| day    | 是   | string | 查询日期 格式：20221001 |

| 参数名 | 描述 | 类型 |
