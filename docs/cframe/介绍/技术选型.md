# 技术选型

**1、系统环境**

- Java EE 8

- Apache Maven 3

**2、主框架**

- Spring Boot 2.7.0
- Spring Framework 2.3.2
- Apache Shiro 1.6.0
- Quartz 2.3.0

**3、持久层**

- [Apache MyBatis-plus 3.3.2](https://baomidou.com/)

- Beetl 1.2.28.RELEASE

- [Hutool 5.8.2](https://hutool.cn/docs/#/)
