# 概述

本 CMS 作为框架中的一个业务功能提供给用户使用。支持自定义模型、自定义表单、文章发布审核、多级用户管理功能。  
拥有一套完整的 CMS 系统功能，可以与系统内的其他模块配合使用，也可独立使用。
