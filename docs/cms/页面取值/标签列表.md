# 标签列表

<Alert type="warning">
	只有列表页面可使用<br/>  
    - tag_xxxx.html   <br/>
</Alert>

| 名称              | 描述          |
| ----------------- | ------------- |
| ${taginfo.title}  | 标签 SEO 标题 |
| ${taginfo.url}    | 标签链接地址  |
| ${taginfo.weight} | 标签权重      |
| ${code}           | 当前专题 code |
| ${current}        | 当前页        |
| ${urlpre}         | 路径前缀      |
| ${pageurl}        | 分页路径前缀  |
| ${currenturl}     | 当前页 URL    |
